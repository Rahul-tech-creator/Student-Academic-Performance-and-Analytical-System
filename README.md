# Student-Academic-Performance-and-Analytical-System# 🎓 Smart Academic Performance & Analytics System (SAPAS)

> Turning raw student marks into meaningful insights — one CSV at a time.

---

## 🚀 Overview

SAPAS is a simple yet powerful Python-based system that manages student academic records using a CSV file.
It performs core operations like adding, updating, deleting, and analyzing student data in a structured way.

This project focuses on:

* Clean design 🧠
* Modular coding ⚙️
* Practical file handling 📂

---

## ✨ Features

* ➕ Add Student Records
* 📄 Display All Students (Tabular Format)
* 🔍 Search Student by ID
* ✏️ Update Student Details
* ❌ Delete Student Record
* 📊 Automatic Grade & CGPA Calculation
* 🗂️ CSV File Auto-Creation

---

---

## ⚙️ How It Works

1. User interacts through a menu-driven interface
2. Data is stored in a CSV file
3. All operations (CRUD) update the file dynamically
4. Output is displayed in a clean tabular format

---

## ▶️ How to Run

```bash
python main.py
```

Make sure you run the program from the **project root folder**.
